# vite-demo-react

使用如下技术栈

- vite - 打包框架
- react - 视图渲染
- react-router - 路由
- react-query - 请求框架
- typescript
- antd - ui 框架
- axios - 请求框架
- lodash-es - 辅助函数
- valtio - 数据流
- eslint - 代码检查
- prettier - 代码格式化
- pnpm - 包管理工具

# 启动/构建

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建打包
pnpm build

# 打包预览
pnpm preview
```

#### node/pnpm 版本

pnpm >= 8.0.0

node >= 16.0.0
